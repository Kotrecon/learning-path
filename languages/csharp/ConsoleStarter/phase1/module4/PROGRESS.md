## ✅ Критерий приёмки модуля

- `IOptionsMonitor<AppSettings>` инжектится в сервис и предоставляет `CurrentValue`
- `OnChange` callback вызывается при изменении `appsettings.json` без перезапуска приложения
- Валидация через `ValidateDataAnnotations` и `ValidateOnStart` блокирует запуск при некорректном конфиге
- Сервис применяет новое значение конфига в рантайме после изменения файла
- `module4/README.md` содержит примеры кода и инструкции по проверке hot-reload
- Все изменения зафиксированы в Git с понятной историей коммитов
