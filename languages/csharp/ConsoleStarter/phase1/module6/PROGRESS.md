## ✅ Критерий приёмки модуля

- `Program.cs` содержит корректный порядок инициализации: конфигурация → опции → DI → хост → запуск
- `ILogger<T>` используется для структурированного логирования в компонентах
- Базовый `try/catch` в `Main` логирует необработанные исключения и возвращает код выхода `1`
- Скрипт/тест на `TUnit` запускает приложение 5 раз, все запуски завершаются с кодом `0`
- `dotnet publish -c Release -r win-x64 --self-contained true` создаёт рабочий билд в `./publish`
- `module6/README.md` содержит инструкции по сборке, запуску, тестированию и публикации
- Все изменения зафиксированы в Git с понятной историей коммитов по всем 6 модулям
