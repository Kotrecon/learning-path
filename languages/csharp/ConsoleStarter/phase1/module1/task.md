# Модуль 1 — .NET 10 Console Modernization

> Цель: Создать базовый `Program.cs` на **.NET 10**, перейти на top-level statements в **C# 14**, использовать `Host.CreateApplicationBuilder(args)` и убрать boilerplate.

## Секция 1: Переход на top-level statements

### Задача 1.1: Мигрировать точку входа на top-level statements в C# 14

**Описание:** Удалить `class Program` и `static void Main`, заменив их на код корневого уровня в **C# 14**. Понять, что компилятор автоматически генерирует точку входа. Убедиться, что проект **.NET 10** компилируется и запускается с минимальным кодом. Инициализировать Git-репозиторий для контроля версий изменений.

**Способы достижения:**

- Создать проект: `dotnet new console -n ModernApp -f net10.0`
- Удалить `class Program` и `static void Main`, оставить `Console.WriteLine("Hello .NET 10");`
- Инициализировать репозиторий: `git init`, `dotnet new gitignore`, выполнить первый коммит
- Проверить сборку: `dotnet build`, при необходимости проверить сгенерированный IL через SharpLab

**Результат:** Файл `Program.cs` с одной строкой кода, инициализированный Git-репозиторий с коммитом `feat(module1): migrate to top-level statements`, успешный вывод `dotnet build`.

## Секция 2: Инициализация `Host.CreateApplicationBuilder(args)`

### Задача 1.2: Настроить базовый хост через `Host.CreateApplicationBuilder(args)` в .NET 10

**Описание:** Подключить `Microsoft.Extensions.Hosting` и инициализировать хост через `Host.CreateApplicationBuilder(args)` в **.NET 10**. Понять, что билдер уже даёт доступ к `builder.Configuration`, `builder.Services` и управляет инфраструктурой приложения. Зафиксировать изменения в репозитории с понятным сообщением коммита.

**Способы достижения:**

- Использовать пакет/ссылку на `Microsoft.Extensions.Hosting`, доступную в **.NET 10**
- В `Program.cs` добавить:

  ```csharp
  var builder = Host.CreateApplicationBuilder(args);
  var host = builder.Build();
  await host.RunAsync();
  ```

- Запустить приложение, проверить завершение по `Ctrl+C` с кодом `0`
- Зафиксировать: `git add . && git commit -m "feat(module1): implement basic host application builder setup"`

**Результат:** `Program.cs` с инициализацией современного хоста **.NET 10**, лог успешного запуска и завершения, коммит с настройкой `HostApplicationBuilder` в истории репозитория.
