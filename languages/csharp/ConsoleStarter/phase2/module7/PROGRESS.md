## Секция 3: Logging Scopes — контекстная группировка

### [x] Задача 7.3: Реализовать логические группы событий через BeginScope

- **Статус:** ✅ Completed
- **Дата выполнения:** 2026-05-29
- **Коммит:** `feat(module7.3): implement logging scopes with IncludeScopes option`
- **Заметки/Наблюдения:**
  - В .NET 10 `JsonConsoleFormatter` требует явного `options.IncludeScopes = true` для вывода контекста
  - Данные scope рендерятся в массив `Scopes`, а не мержатся в `State` (стандарт поведения провайдера)
  - 💡 Инсайт: Scope гарантирует инъекцию контекста без передачи параметров. В продакшене это исключает потерю `CorrelationId` при рефакторинге.

- **Статус:** ✅ Completed
- **Дата выполнения:** 2026-05-29

- **Заметки/Наблюдения:**
